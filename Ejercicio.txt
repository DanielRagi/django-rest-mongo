Realizar el registro, consulta, modificación y eliminación de registros de películas a través de un Web Service con comunicación HTTP hacia una API de tipo REST. Para cada película se almacenará un ID numérico automático, nombre, género, descripción, director y año, y se almacenará en una base de datos no relacional en MongoDB. Se creará la vista para consulta, modificación y eliminación y la vista para el registro. La API se debe exponer en internet
